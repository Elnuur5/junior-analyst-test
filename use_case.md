# Use Case: Публикация товара

## Актор:
Продавец

## Предусловия:
Пользователь авторизован в системе

## Основной сценарий:
1. Продавец открывает форму добавления товара
2. Заполняет данные товара
3. Загружает изображения
4. Выбирает категорию
5. Нажимает кнопку "Опубликовать"
6. Система проверяет корректность данных
7. Система сохраняет товар
8. Товар появляется на витрине

## Альтернативные сценарии:

- Если название пустое → ошибка
- Если цена ≤ 0 → ошибка
- Если нет изображения → предупреждение/ошибка
- Если пользователь не авторизован → отказ в доступе
