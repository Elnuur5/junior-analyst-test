# Functional Requirements

1. Система должна позволять продавцу создавать товар.
2. Система должна позволять указывать название товара.
3. Система должна позволять указывать описание товара.
4. Система должна позволять указывать цену товара.
5. Система должна позволять загружать изображения товара.
6. Система должна позволять выбирать категорию товара.
7. Система должна публиковать товар на витрине после сохранения.
